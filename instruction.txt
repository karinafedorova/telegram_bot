
Инструкция

Чтобы запустить бота откройте Ваш терминал и выполните следующие команды:
1. docker build .
2. docker images (это команда отобразит Ваш 'IMAGE ID')
3. docker run -d -p 80:80 'IMAGE ID'
4. docker ps (это команда выведет Ваши активные контейнеры и 'id')


Чтобы отключить запущенный контейнер нужно ввести следующую команду:
docker stop "id"
